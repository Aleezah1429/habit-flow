# Mission — HabitFlow

## What we're building
**HabitFlow** is a minimal, fast habit tracker that helps users build daily routines by visualizing streaks and consistency over time.

## Why
Most habit apps are bloated with social features, gamification, or paywalls. HabitFlow stays focused on the one job that matters: **mark a habit done today, see your streak grow.**

## Target user
- Individuals who want to build 3–10 daily habits (e.g. "drink 2L water", "read 20 mins", "no sugar").
- Comfortable with a clean web UI; no need for mobile app v1.

## Core value
1. **Friction-free logging** — one click to mark a habit done today.
2. **Streak visibility** — current streak is the dopamine hit; users come back for it.
3. **Honest history** — heatmap shows real consistency, not vanity metrics.

## Non-goals (v1)
- No multi-user, no auth, no sync (local-storage based).
- No social sharing, no friends, no leaderboards.
- No mobile native app.
- No reminders/notifications (browser-based only — out of scope).

## Success criteria
- A user can add a habit, mark it done for 7 consecutive days, and see a 7-day streak — all without reading docs.
- Page loads in under 1 second.
- Zero data loss across browser refreshes.
